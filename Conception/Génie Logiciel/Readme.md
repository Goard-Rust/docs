| [Diagramme de Contexte](#diagramme-de-contexte) | Vue globale du système et ses interactions externes |
| [Vue de Haut Niveau](#vue-de-haut-niveau) | Architecture générale du système |
| [Vue Détaillée - View](#vue-détaillée---view) | Composants de la couche View (MVC) |

| [Vue Système](#vue-système) | Interactions système avec l'environnement extérieur |
